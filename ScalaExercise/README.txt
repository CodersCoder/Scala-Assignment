To run different problem statements:

1. AadhaarScala code :
   Pass the arguments as 
   	1. input file path of auth file
   	2. output folder path

1. AadhaarSpark code :
   Pass the arguments as 
   	1. input file path of auth file
   	2. output folder path

1. PovertyReport code :
   Pass the arguments as 
   	1. input file path of Poverty Estimates excel file
   	2. input file path of States name excel file
   	3. output folder path
